## Prerequisites

- [Node.js](https://nodejs.org/en/download) 8 or higher is required.
- [MongoDB](https://www.mongodb.com/download-center) 3 or higher is required.

## Installation

```bash
npm i -s @foxify/odin
```

## Usage

TypeScript

```typescript
import * as Odin from "@foxify/odin";
```

JavaScript

```javascript
const Odin = require("@foxify/odin");
```
